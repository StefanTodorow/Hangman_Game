Rules of Hangman game

The rules are simple

  1. You start the game with 3 lives.

  2. Think of one letter that can fit in the word.

  3. Insert your letter in the textbox below the hidden word.

  4. You can't insert letter that is already used.

  5. Click ENTER or click on the button to try the letter.

  6. If you guessed wrong, you will lose 1 life.

  7. You win if you guessed the word and lose if you drop down to 0 lives(3 wrong letters).

  8. Enjoy!

If you have any issues with the game or want to report a bug, please contact me in the Contacts page. Thank you!
